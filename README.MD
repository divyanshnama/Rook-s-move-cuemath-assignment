#  Tech Stack and Approaches

JavaScript, PhaserJS 3.6, Vite, NodeJS, ExpressJS, Socket.IO

#   Run Your Own RookMove Instance ?

Install client and server dependencies

```bash

$ npm install
$npm run dev
$ cd server
$ npm install
$npm start

follow the link after all the steps ( http://localhost:5173/ ) to access your instance of RookMove! and duplicate the tab for other player
```
